

Concrete Creator:
    the client that calls the creator (the factory method)

Product Interface:
    describes the things that the factory requires in order to create things.
    'factory method knows this "THING" in order to work with the products'

Creator:
    the factory class. declares the factory method and returns the objects required from it

Concrete Product:
    the objects that the factory method will return, this guy implements the product interface
